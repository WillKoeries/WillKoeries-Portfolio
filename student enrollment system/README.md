# Student Enrollment System

## Description
The Student Enrollment System is a Java-based application that allows users to log in as either an administrator or a student. The system provides role-based access, enabling administrators to manage data and students to enroll in courses and view their enrollments.

## Features

### Admin Features
- Add new students, courses, and users
- View existing records
- Update student, course, and enrollment information
- Delete students, courses, enrollments, and users

### Student Features
- Log in to a personal dashboard
- Enroll in available courses
- View enrolled courses

## Technologies Used
- Java
- Netbeans (or any Java IDE)

## Concepts Implemented
- Conditional statements (if/else)
- User roles and access control
- CRUD operations (Create, Read, Update, Delete)
- Basic input handling and validation

## Installation
1. Download or clone the repository
2. Open the project in IntelliJ IDEA (or any Java IDE)
3. Navigate to the **server** folder
4. Run **ServerApp.java** to start the server
5. Navigate to the **GUI** folder
6. Run **LoginGUI.java** to launch the application interface

## Usage
- First, run **ServerApp.java** to start the backend server
- Then, run **LoginGUI.java** to open the application
- Log in as either:
  - **Admin** to manage students, courses, enrollments, and users
  - **Student** to enroll in courses and view your enrollments
- Use the on-screen options to interact with the system
