# Community Impact website

## Description
The Community Impact Initiatives NGO is a web-based application that showcases a non-profit organization's efforts in education, health, and environmental sustainability. The website provides information about the organization's mission, projects, and ways for community members to get involved through volunteering and donations.

**Note:** This was a group project completed as part of CPUT coursework.

## Features

### Website Sections
- **Home Page** - Overview of the organization's impact and initiatives
- **About Us** - Mission, vision, history, and social media links
- **Projects** - Detailed information on education, health, and environmental projects
- **Volunteer** - Sign-up form for community volunteers
- **Donation** - Form for financial contributions
- **Contact Us** - Contact form for inquiries

### Interactive Features
- Image sliders on project pages
- Collapsible content sections
- Form data export to text files
- Responsive design for mobile devices
- Smooth CSS animations

## Technologies Used
- HTML
- CSS
- JavaScript 

## Concepts Implemented
- Responsive web design
- CSS animations and transitions
- DOM manipulation
- Form handling and validation
- Local file generation 
- Event listeners

## Installation
1. Download or clone the repository
2. Navigate to the **community-impact-website** folder
3. Open **HomePage.html** in any web browser
4. No server required - runs entirely in browser

## Usage
- Open **HomePage.html** to view the landing page
- Navigate using the menu bar:
  - **About Us** - Learn about the organization
  - **Projects** - View education, health, and environment initiatives
  - **Volunteer** - Fill out the volunteer sign-up form
  - **Donate** - Make a donation through the form
  - **Contact Us** - Send a message to the organization
- Forms automatically download data as text files when submitted

## Project Structure
- **Css/** → Stylesheets for all pages
- **Js/** → JavaScript files for interactivity
- **Images/** → Logos and project images
- **[HTML files]** → 9 HTML pages
